Orac REST API:

REST base address:
http://<host>:8080/Orac/rest/service/

Get user:
get_user/{user_name}
 
Get all users:
get_users

Add user:
add_user/{user_name}

Add user with description:
add_described_user/{user_name}/{description}

Delete user:
delete_user/{user_name}

Get resource:
get_resource/{resource_name}

Get all resources:
get_resources

Add resource.
add_resource/{resource_name}

Add a resource with description:
add_described_resource/{resource_name}/{description}

Delete resource:
delete_resource/{resource_name}

Recommend new friends for user:
recommend_friends/{user_name}/{max_friends}

Recommend new resources for user:
recommend_resources/{user_name}/{max_resources}

Get user friends:
get_friends/{user_name}

Befriend user:
befriend_user/{user_name}/{friend_name}
   
Unfriend user:
unfriend_user/{user_name}/{friend_name}

Unfriend all:
unfriend_all/{user_name}

Rate resource:
rate_resource/{user_name}/{resource_name}/{resource_rating}
